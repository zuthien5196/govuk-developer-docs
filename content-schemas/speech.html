This schema describes the content item that is returned from the content store.

string or null		 A short identifier we send to Google Analytics for multi-valued fields. This means we avoid the truncated values we would get if we sent the path or slug of eg organisations.

string		 A path only. Query string and/or fragment are not allowed.
